Configurações

Integrações e estrutura técnica

Google Sheets

As avaliações são enviadas automaticamente para a planilha via Google Apps Script. Configure a URL do seu Web App na constante URL_GOOGLE_SHEETS no código-fonte. Enquanto não configurado, os dados ficam salvos localmente no navegador (localStorage).

Notificação via WhatsApp

Estrutura pronta para envio automático de notificação ao número da equipe +55 47 99166-5148 a cada nova avaliação. A função notificarWhatsApp() está pronta para ser conectada a uma API (Evolution API, Meta WhatsApp Cloud API, ou similar) — basta inserir a URL do endpoint e o token de autenticação nas constantes no topo do código. Até a configuração, a notificação é apenas registrada no console e fica disponível como link manual de WhatsApp no painel.
